# Скриншоты и подсказки

Пока в репозитории нет готового набора скриншотов, но ниже есть список того, что стоит снять для docs.

Если ты используешь `clawdbot`, можно пройтись по этому чек-листу и постепенно наполнить документацию визуалами.

## 1. Главный экран desktop

Что снять:

- экран `Проект`
- заполненное поле задачи
- видимые карточки с состоянием проекта

Подсказка для Clawdbot:

> Открой ARC Desktop на экране Проект. Покажи понятный интерфейс для обычного пользователя: слева навигация, в центре карточка проекта, ниже поле для новой задачи и простые кнопки “Сделать план” и “Попробовать безопасно”.

## 2. Экран `Задачи`

Что снять:

- список задач
- выбранную задачу
- блок с изменёнными файлами

Подсказка для Clawdbot:

> Покажи экран ARC Desktop “Задачи”: слева список задач, справа подробности выбранной задачи, статусы, кнопка “Обсудить в чате”, список изменённых файлов и понятные русские подписи.

## 3. Экран `Чат`

Что снять:

- список чатов
- открытую переписку
- видимый блок добавленного контекста

Подсказка для Clawdbot:

> Покажи экран ARC Desktop “Чат” для обычного пользователя: слева список чатов, справа переписка с агентом, наверху короткая форма нового чата, рядом блок “Что добавить в контекст”.

## 4. Экран `Наборы`

Что снять:

- список наборов
- preview изменений
- понятные кнопки установки

Подсказка для Clawdbot:

> Покажи экран ARC Desktop “Наборы”: слева библиотека готовых наборов, справа preview изменений, статус конфликтов и понятные кнопки “Установить” и “Добавить в контекст”.

## 5. Экран `Файлы`

Что снять:

- список файлов
- открытый файл
- diff

Подсказка для Clawdbot:

> Покажи экран ARC Desktop “Файлы”: слева список файлов проекта, справа содержимое выбранного файла, блок с diff и простые кнопки “Сохранить файл” и “В новую задачу”.
