# Первый сценарий от начала до конца

Этот walkthrough показывает один полный рабочий цикл.

## Сценарий

Представим, что ты хочешь:

> посмотреть проект, найти следующее безопасное улучшение и получить понятный результат

## Вариант через CLI

### 1. Подготовка

```bash
./bin/arc doctor
./bin/arc init --path .
./bin/arc mode set work --path .
```

### 2. План

```bash
./bin/arc task plan --path . --mode work --provider codex "Посмотри проект и предложи следующее безопасное улучшение"
```

### 3. Безопасный запуск

```bash
./bin/arc task run --path . --mode work --provider codex --dry-run "Подготовь следующее безопасное улучшение"
```

### 4. Посмотреть статус

```bash
./bin/arc run list --path .
./bin/arc run status --path .
```

### 5. Если нужно продолжить

```bash
./bin/arc chat start --path . --provider codex --mode work "Продолжи эту задачу и предложи следующий шаг"
```

## Вариант через Desktop

### 1. Открой экран `Проект`

Сначала выбери папку проекта.

Если в папке ещё нет `.arc`, нажми `Настроить ARC`.

### 2. Напиши задачу:

> Посмотри проект и предложи следующее безопасное улучшение

### 3. Нажми `Сделать план`

### 4. Перейди на экран `Задачи`

Там появится новая задача.

### 5. Посмотри результат

Обрати внимание на:

- статус
- отчёты
- изменённые файлы

### 6. Если нужно, нажми `Обсудить в чате`

Так ты продолжишь работу не с нуля, а уже от существующего результата.

## Что считается хорошим результатом

После первого сценария у тебя должны быть:

- созданный `.arc/`
- хотя бы одна задача
- хотя бы один набор артефактов
- понятное ощущение, где в ARC начинать работу дальше
